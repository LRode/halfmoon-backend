# Halfmoon Backend

This project is using Strapi Headless CMS

## Prequisites

Install `npm` and `node`

Software	Minimum version
Node.js	    12.x
npm	     6.x


## Installation

```
npm install
```

## Running Strapi

```
npm run strapi develop
```

## Running Cloudinary

```
npm i strapi-provider-upload-cloudinary
```
